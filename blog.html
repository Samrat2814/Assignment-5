Q: What is the DOM in JavaScript?

A: The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects that can be manipulated with JavaScript.

Q: What is the difference between innerHTML and textContent?

A:
1. innerHTML can set or get HTML content.
2. textContent sets or gets plain text, ignoring HTML tags.

Q: How can you append an element to the DOM?

A:
Use parentElement.appendChild(childElement).
Example:
const parent = document.getElementById("container");
const child = document.createElement("p");
parent.appendChild(child);

Q: How do you update the text content of an element?

A:
Use the .textContent property.
Example:
const element = document.getElementById("myId");
element.textContent = "Updated Text";
